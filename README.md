## Getting Started

### Installation

1. Clone the repo to local:
```
git clone https://github.com/andrewlin618/World-Channel.git
cd World-Channel
```

2. Install the dependencies:
* express
* concurrently
* socket.io
* nodemon
```
npm install
```
### Usage
1. Run the app with:
```
npm start
```
2. Open http://localhost:3001 to take a look around
